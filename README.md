Sistema de Adoção Pet
Uma aplicação completa que simula o fluxo de uma ONG, desde o cadastro do animal até a formalização da ficha de adoção.

### Funcionalidades Chave
* **Formulários Inteligentes:** Captura de dados de usuários e pets com validação via JavaScript.
* **Renderização Dinâmica:** Exibição imediata das informações do usuário após a conclusão do cadastro.
* **Contador em Tempo Real:** Sistema lógico que incrementa e exibe o total de registros processados (+1).
* **Integração com Banco de Dados:** Persistência de dados real utilizando **MySQL** para armazenar as fichas e métricas.

## Tecnologias Utilizadas
* **Frontend:** HTML5 Semântico, CSS3 (Responsividade) e JavaScript Vanilla (Manipulação de DOM).
* **Backend & Servidor:** PHP e Ambiente **XAMPP** (Apache).
* **Banco de Dados:** MySQL / MariaDB.
* **Versionamento:** Git & GitHub.
